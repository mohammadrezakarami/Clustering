# Clustering

This repository contains unsupervised learning notebooks focused on clustering customer data.

## Repository Contents

- `k_mean.ipynb`: K-Means clustering workflow.
- `hierarchical.ipynb`: hierarchical clustering workflow.
- `Mall_Customers.csv`: dataset used in the clustering experiments.

## Topics Covered

- unsupervised learning
- K-Means clustering
- hierarchical clustering
- customer segmentation

## Getting Started

1. Clone the repository.
2. Open the notebooks in Jupyter Notebook or JupyterLab.
3. Install the standard Python data science stack.
4. Run each notebook to reproduce the clustering analysis.

## Notes

This repository is a compact educational collection of clustering implementations on a common benchmark-style dataset.
